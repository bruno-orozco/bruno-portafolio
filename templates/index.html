Quién es Bruno Orozco

Bruno Orozco es un economista y programador. Nació el 24 de Septiembre de 1998 en México. Realizó sus estudios en la Facultad de Economía de la UNAM. Le encanta el universo tecnológico. Le apasiona el Desarrollo Web, la Ciencia de Datos y la Inteligencia Artificial.

Su lenguaje de programación favorito es Python. En él, se divierte elaborando aplicaciones con Django, Flask, Pandas, Numpy, Matplotlib y Plotly. Domina los paradigmas de Programación Orientada a Objetos y Programación Funcional.

No obstante, también te puede ayudar a crear proyectos en Excel y R studio. Además, sabe implementar bases de datos SQL, y data viz con Power BI, Tableau, y Google Data Studio.

Conoce sus proyectos principales.

Micro-sofware: creador de polizas

Este micro-software es un creador de pólizas contables. En la empresa se tenía el siguiente problema: se invertían muchas horas de tiempo para realizar cálculos básicos. Este programa identifica automáticamente si el monto monetario es un cargo o un abono. Además, toma en cuenta la sucursal y la fecha en la que se realizó la transacción.

Este script fue hecho 100% en Python. Utilizó la librería pandas para la manipulación de datos, y tkinter para obtener una interfaz gráfica amigable para cualquier usuario. Con esto, se consiguió que cualquier persona pueda utilizar el código sin la necesidad de tener grandes conocimientos computacionales. La cabecera prevé los posibles errores que podría cometer el usuario, proporcionando instrucciones sobre qué hacer para solucionarlo.

App web: Gestión de Relación con el Cliente

Esta app, surgió de la ncesidad de recolectar los datos que los vendedores de la empresa generaban. Ya que los empleados pertenecían a diferentes regiones de la república mexicana, había un problema de supervisión de la actividad diaria. Este web-site resolvió muchos de estos problemas. Por un lado, el empleado podía ver el avance de sus métricas de productivadad en tiempo real. Con ello, se logró detectar las áreas de oportunidad en el proceso de prospección.

Para la creación de esta app se recurrió a HTML, CSS y Javascript para generar el Frontend. Para el Backend, se utilizó el framework Django de Python. Las bases de datos están compiladas con PostreSQL. Además, se implementó la automatización de reportes diarios vía email a los líderes de ventas. Se recurrió a las librerías psycopg2, pandas, y win32com para este propósito.

Cursos de programación: Análisis económico con Python

A medidados de este año, impartió cursos de programación con enfoque económico. La idea surgió a partir de identificar que no existía un curso de este género. La diferenciación principal respecto a otros cursos, consitió en guíar las herramientas necesarias para que un aspirante a economista profesional pueda aprovechar este lenguaje y encontrar un sentido práctico.

Estudiantes de economía tanto de nivel maestría como de los primeros semestres de economía reseñaron positivamente este curso.

Su trabajo pedagógico también se encuentra en YouTube. Esta es una de las clases públicas de estadística con Python.
